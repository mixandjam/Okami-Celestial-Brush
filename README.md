## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

-  Unity ver. 2019.4 or higher

### Running the project

After cloning or downloading the repository, open the root project folder in unity and press the Play button.

You can see a running demo of the project in [this video](https://youtu.be/yuQXeaYBuuM).

## Built With

* [DOTween](http://dotween.demigiant.com/) - DOTween is a fast, efficient, fully type-safe object-oriented animation engine for Unity, optimized for C# users, free and open-source, with tons of advanced features
